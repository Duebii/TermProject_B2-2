# Team 18 | AI 기술 뉴스 자동 수집·요약 시스템

## 프로젝트 소개

본 프로젝트는 RSS 피드를 활용하여 최신 AI 기술 뉴스를 자동 수집하고, 생성형 AI를 통해 핵심 내용을 요약한 후 노션 데이터베이스에 자동 저장하는 자동화 시스템입니다.

뉴스 수집부터 AI 요약, 데이터 저장까지의 전 과정을 자동화하여 반복적인 정보 탐색 업무를 줄이고 효율적인 기술 동향 파악이 가능하도록 설계하였습니다.

---

## 프로젝트 목표

- RSS 기반 AI 기술 뉴스 자동 수집
- 생성형 AI를 활용한 3줄 요약 생성
- 노션 데이터베이스 자동 저장
- 중복 저장 방지
- 에러 처리 적용
- 스케줄 기반 자동 실행

---

## 사용 기술

| 구분 | 사용 도구 |
| --- | --- |
| 자동화 | n8n, Make |
| AI | Gemini API |
| 저장소 | Notion |
| 데이터 수집 | RSS Feed |
