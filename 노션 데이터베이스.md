- **(Case 2) 중복 기사**
자동화 전 테스트 중 뉴스 중복 발견 → n8n Code in JavaScript에 설정된 값
“항상 최신 기사 1개 선택” 말고 새로 추가로 설정함
이미 저장한 GUID 제외 → AI 관련성 높은 기사 선택 → 그 중 최신 기사 선택 
→ 없으면 중복되지 않은 다른 최신 기사 선택 → 전부 중복이면 ‘저장하지 않음’으로 변경
